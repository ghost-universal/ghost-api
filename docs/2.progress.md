## Implementation Guidelines

### Pattern Used

All methods follow the scaffolding pattern:

```rust
pub fn some_method(&self, ...) -> Result<T, GhostError> {
    // TODO: Implement some_method
    // Actual implementation goes here
    Err(GhostError::NotImplemented("some_method".into()))
}
```

### Type Definitions

All types have:
- Complete field definitions
- Constructor methods with TODO comments
- Builder patterns where appropriate
- Validation methods (returning Ok(()) by default)

### No Business Logic

- All implementations return `NotImplemented` errors
- Data structures are complete and properly typed
- Serialization/Deserialization is configured via serde
- All imports are properly declared
