## 1. Executive Summary

This document defines a **comprehensive polyglot architecture** for `ghost-api` that enables seamless integration of multiple external scrapers (Python, Node.js, Go) with **git-based update workflows**. The architecture transforms external repositories into standardized GhostWorkers without forking or manual synchronization.

### Strategic Goals

| Goal | Problem | Solution |
|------|---------|----------|
| **Multi-Polyglot Support** | Many external scrapers needed | Unified interface & plugin system |
| **Easy Updates** | Manual sync is error-prone | Git submodules + automated pull |
| **Capability Discovery** | Unknown scraper capabilities | Standardized manifest.json |
| **Dependency Isolation** | Conflicting Python versions | Per-polyglot virtual environments |
| **Health Integration** | No unified health monitoring | GhostWorker trait implementation |

### Key Design Decisions

1. **Git Submodules**: External scrapers integrated as git submodules in `scrapers/external/`
2. **Manifest-Driven**: Each polyglot exports `manifest.json` declaring capabilities
3. **Worker Wrapper**: Thin Python wrapper translates scraper output to GhostWorker interface
4. **PyO3 Bridge**: Rust calls Python workers via PyO3 FFI layer
5. **Health Scoring**: Each polyglot integrates with ghost-core health system

## 3. Git Submodule Strategy

### 3.1 Submodule Architecture

The polyglot system uses **git submodules** to enable easy updates from upstream repositories. This approach provides:

- **One-command updates**: `git submodule update --remote`
- **Version pinning**: Lock to specific commits/tags
- **Isolation**: External code doesn't pollute main repo
- **Attribution**: Original authors maintained

### 3.2 Current Submodule Setup

```bash
# Already cloned
scrapers/external/threads-scraper/
├── threads_scraper.py       # v1 basic (125 lines)
├── threads_scraper_v2.py    # v2 enhanced (362 lines)
├── README.md
└── LICENSE                  # MIT License
```

### 3.3 Submodule Configuration (.gitmodules)

```ini
[submodule "threads-scraper"]
    path = scrapers/external/threads-scraper
    url = https://github.com/vdite/threads-scraper.git
    branch = main
    fetchRecurseSubmodules = true
    update = merge

# Future submodules

    path = scrapers/external/twitter-scraper
```

### 3.4 Update Workflow

#### Automated Update Script

```bash
#!/bin/bash
# scripts/sync-polyglots.sh

set -e

echo "🔄 Syncing polyglot submodules..."

# Update all submodules to latest
git submodule update --init --recursive
git submodule update --remote --merge

# Check for changes
for submodule in scrapers/external/*/; do
    name=$(basename "$submodule")
    cd "$submodule"
    
    # Get current commit
    current=$(git rev-parse HEAD)
    upstream=$(git rev-parse @{u})
    
    if [ "$current" != "$upstream" ]; then
        echo "📦 $name has updates available"
        echo "   Current: $current"
        echo "   Upstream: $upstream"
    fi
    
    cd - > /dev/null
done

echo "✅ Polyglot sync complete"
```

#### Version Pinning Strategy

```bash
# Pin to specific version (recommended for production)
cd scrapers/external/threads-scraper
git checkout v2.0.0  # or a specific commit hash
cd ../..
git add scrapers/external/threads-scraper
git commit -m "chore: pin threads-scraper to v2.0.0"

# Update to latest in development
git submodule update --remote scrapers/external/threads-scraper
```

### 3.5 Wrapper Layer Design

The wrapper layer isolates the external scraper from ghost-api internals:

```
scrapers/py-threads/
├── wrapper.py              # GhostWorker interface → calls external/
├── manifest.json           # Capability declaration
├── requirements.txt        # Dependencies (playwright, parsel, nested-lookup, jmespath)
└── pyproject.toml          # Build config

scrapers/external/threads-scraper/   # Git submodule (untouched)
├── threads_scraper.py      # v1 - 125 lines
├── threads_scraper_v2.py   # v2 - 362 lines
└── README.md
```

**Key principle**: External submodule code is **never modified**. All adaptations happen in the wrapper layer.

## 4. Standardized Polyglot Interface

### 4.1 GhostWorker Python Interface

All Python polyglots must implement this standardized interface:

```python
# scrapers/py-threads/wrapper.py

import json
import sys
import os
from typing import Dict, List, Optional, Any
from dataclasses import dataclass, asdict
from abc import ABC, abstractmethod
import asyncio

# Add external submodule to path
EXTERNAL_PATH = os.path.join(os.path.dirname(__file__), '..', 'external', 'threads-scraper')
sys.path.insert(0, EXTERNAL_PATH)


@dataclass
class GhostContext:
    """Unified context for all polyglot workers"""
    tenant_id: Optional[str] = None
    cookies: Optional[str] = None          # Cookie string or file path
    proxy: Optional[str] = None            # Proxy URL (socks5:// or http://)
    user_agent: Optional[str] = None
    metadata: Dict[str, str] = None
    
    def __post_init__(self):
        if self.metadata is None:
            self.metadata = {}
    
    def to_dict(self) -> Dict:
        return asdict(self)


@dataclass
class PayloadBlob:
    """Raw data returned by scrapers"""
    data: bytes                           # Raw content (JSON, HTML, etc.)
    content_type: str                     # MIME type
    source_url: str                       # Original URL
    status_code: int                      # HTTP status
    headers: Dict[str, str] = None        # Response headers
    metadata: Dict[str, Any] = None       # Additional metadata
    
    def __post_init__(self):
        if self.headers is None:
            self.headers = {}
        if self.metadata is None:
            self.metadata = {}
    
    def to_json(self) -> str:
        """Serialize to JSON for FFI transfer"""
        return json.dumps({
            'data': self.data.decode('utf-8') if isinstance(self.data, bytes) else self.data,
            'content_type': self.content_type,
            'source_url': self.source_url,
            'status_code': self.status_code,
            'headers': self.headers,
            'metadata': self.metadata,
        })


@dataclass
class WorkerHealth:
    """Health status for a worker"""
    status: str                           # 'healthy', 'degraded', 'unhealthy'
    score: float                          # 0.0 - 1.0
    latency_ms: int                       # Average latency
    success_rate: float                   # Recent success rate
    last_check: str                       # ISO timestamp
    message: Optional[str] = None


class GhostWorkerBase(ABC):
    """Base class for all polyglot workers"""
    
    @property
    @abstractmethod
    def id(self) -> str:
        """Unique worker identifier"""
        pass
    
    @property
    @abstractmethod
    def capabilities(self) -> List[str]:
        """List of capabilities this worker provides"""
        pass
    
    @property
    @abstractmethod
    def platforms(self) -> List[str]:
        """List of platforms this worker supports"""
        pass
    
    @abstractmethod
    def execute(self, ctx: GhostContext) -> PayloadBlob:
        """Execute a scraping request"""
        pass
    
    def health_check(self) -> WorkerHealth:
        """Check worker health status"""
        # TODO: Implement health check
        return WorkerHealth(
            status='unknown',
            score=0.0,
            latency_ms=0,
            success_rate=0.0,
            last_check='',
        )
    
    def validate_context(self, ctx: GhostContext) -> bool:
        """Validate the context for this worker"""
        return True


class ThreadsWorker(GhostWorkerBase):
    """
    GhostWorker implementation for threads-scraper.
    
    Wraps the external threads-scraper submodule to provide
    standardized GhostWorker interface.
    """
    
    def __init__(self):
        self._cookies_path = os.path.join(
            os.path.dirname(__file__), 
            'threads_cookies.json'
        )
    
    @property
    def id(self) -> str:
        return "py-threads-v2"
    
    @property
    def capabilities(self) -> List[str]:
        return ["threads_read", "threads_deep_scrape"]
    
    @property
    def platforms(self) -> List[str]:
        return ["threads"]
    
    def execute(self, ctx: GhostContext) -> PayloadBlob:
        """
        Execute scraping request using threads-scraper.
        
        Args:
            ctx: GhostContext with:
                - metadata['url']: Threads URL to scrape
                - metadata['max_pages']: Max pages to visit (default: 100)
                - cookies: Optional cookie string or file path
                - proxy: Optional proxy URL
        
        Returns:
            PayloadBlob with JSON array of scraped posts
        """
        # Import from external submodule
        from threads_scraper_v2 import scrape_threads_recursive
        
        # Extract parameters from context
        url = ctx.metadata.get('url')
        if not url:
            raise ValueError("URL required in context.metadata['url']")
        
        max_pages = ctx.metadata.get('max_pages', 100)
        
        # Handle cookies
        if ctx.cookies:
            # Write cookies to expected location for threads-scraper
            with open(self._cookies_path, 'w') as f:
                f.write(ctx.cookies)
        
        # Handle proxy configuration
        # TODO: Pass proxy to Playwright context
        
        # Execute scraper (async function)
        results = asyncio.run(scrape_threads_recursive(url, max_pages=max_pages))
        
        # Return as PayloadBlob
        return PayloadBlob(
            data=json.dumps(results).encode('utf-8'),
            content_type='application/json',
            source_url=url,
            status_code=200,
            metadata={
                'scraper': 'threads-scraper-v2',
                'max_pages': max_pages,
                'posts_found': len(results),
            }
        )
    
    def health_check(self) -> WorkerHealth:
        """
        Check if scraper is operational.
        
        Verifies:
        - Playwright/Chromium is installed
        - Cookies are valid (if auth)
        - Can make a test request
        """
        # TODO: Implement health check
        # 1. Check if playwright chromium is installed
        # 2. If cookies exist, verify sessionid is present
        # 3. Optionally make a small test request
        
        return WorkerHealth(
            status='unknown',
            score=0.0,
            latency_ms=0,
            success_rate=0.0,
            last_check='',
        )


# =============================================================================
# FFI Entry Points (called from Rust via PyO3)
# =============================================================================

def execute_worker(context_json: str) -> str:
    """
    Main entry point called from Rust via PyO3.
    
    Args:
        context_json: JSON-serialized GhostContext
    
    Returns:
        JSON-serialized PayloadBlob
    """
    try:
        # Parse context
        ctx_dict = json.loads(context_json)
        ctx = GhostContext(**ctx_dict)
        
        # Create worker and execute
        worker = ThreadsWorker()
        result = worker.execute(ctx)
        
        return result.to_json()
    except Exception as e:
        # Return error as PayloadBlob
        error_blob = PayloadBlob(
            data=json.dumps({'error': str(e)}).encode('utf-8'),
            content_type='application/json',
            source_url='',
            status_code=500,
            metadata={'error': True}
        )
        return error_blob.to_json()


def get_worker_info() -> str:
    """Return worker metadata as JSON (for discovery)"""
    worker = ThreadsWorker()
    return json.dumps({
        'id': worker.id,
        'capabilities': worker.capabilities,
        'platforms': worker.platforms,
    })


def health_check() -> str:
    """Return health status as JSON"""
    worker = ThreadsWorker()
    health = worker.health_check()
    return json.dumps(asdict(health))
```

### 5.2 Schema Validation (Rust)

```rust
// crates/ghost-schema/src/manifest.rs

use serde::{Deserialize, Serialize};
use std::collections::HashMap;

/// Polyglot manifest structure
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct PolyglotManifest {
    /// Unique identifier
    pub id: String,
    /// Human-readable name
    pub name: String,
    /// Semantic version
    pub version: String,
    /// Description
    pub description: String,
    /// Runtime configuration
    pub runtime: RuntimeConfig,
    /// External source configuration
    pub external: ExternalSource,
    /// Capabilities provided
    pub capabilities: Vec<CapabilityDefinition>,
    /// Supported platforms
    pub platforms: Vec<String>,
    /// Dependencies
    pub dependencies: HashMap<String, String>,
    /// Configuration schema
    pub configuration: HashMap<String, ConfigField>,
    /// Health configuration
    pub health: HealthConfig,
    /// Rate limit configuration
    pub rate_limits: RateLimitConfig,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RuntimeConfig {
    /// Runtime type: python, nodejs, go
    #[serde(rename = "type")]
    pub runtime_type: String,
    /// Version constraint
    pub version: String,
    /// Entrypoint (module:class or file:function)
    pub entrypoint: String,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ExternalSource {
    /// Source type: git_submodule, local, pip, npm
    #[serde(rename = "type")]
    pub source_type: String,
    /// Path to external source
    pub path: String,
    /// Upstream URL
    pub upstream: String,
    /// License
    pub license: String,
    /// Attribution
    pub attribution: String,
    /// Files included
    #[serde(default)]
    pub files: Vec<String>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct CapabilityDefinition {
    /// Capability name
    pub name: String,
    /// Human description
    pub description: String,
    /// Tier (1 = fast, 2 = heavy, 3 = fallback)
    pub tier: u8,
    /// Health weight for this capability
    pub health_weight: f64,
    /// Parameter schema
    pub parameters: HashMap<String, ParameterDef>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ParameterDef {
    #[serde(rename = "type")]
    pub param_type: String,
    pub required: bool,
    #[serde(default)]
    pub default: Option<serde_json::Value>,
    #[serde(default)]
    pub description: String,
```

```python
async def scroll_and_expand(page, extracted_comments: Dict, max_stale_rounds: int = 12):
    """Scrolls and clicks load-more buttons until no new content appears."""
    
    load_more_selectors = [
        "text=/View more repli/i",
        "text=/Show hidden repli/i",
        "text=/View all repli/i",
        "text=/more repli/i",
        "text=/Load more/i",
        "text=/See more/i",
        "text=/Show replies/i",
        "text=/View replies/i",
        # German locale support
        "text=/Weitere Antworten/i",
        "text=/Mehr anzeigen/i",
        "text=/Antworten anzeigen/i",
    ]

    previous_count = len(extracted_comments)
    previous_scroll_height = await get_scroll_height(page)
    stale_rounds = 0

    while stale_rounds < max_stale_rounds:
        # Scroll down
        await page.mouse.wheel(0, 4000)
        await page.wait_for_timeout(1500)

        # Click any "load more" / "show replies" buttons
        for sel in load_more_selectors:
            try:
                buttons = await page.locator(sel).all()
                for button in buttons:
                    try:
                        if await button.is_visible(timeout=500):
                            await button.click(timeout=2000)
                            await page.wait_for_timeout(2000)
                            stale_rounds = 0  # reset on successful click
                    except Exception:
                        pass
            except Exception:
                pass

        # Check for progress
        current_count = len(extracted_comments)
        current_scroll_height = await get_scroll_height(page)

        if current_count > previous_count or current_scroll_height > previous_scroll_height:
            previous_count = current_count
            previous_scroll_height = current_scroll_height
            stale_rounds = 0
        else:
            stale_rounds += 1
```

```python
async def do_login(pw):
    """Opens a visible browser for manual login. Saves cookies afterwards."""
    print("\n  === LOGIN MODE ===", file=sys.stderr)

    logged_in = False
    for attempt in range(300):  # up to 5 minutes (300 x 1s)

        cookie_names = [c["name"] for c in cookies]

        if "sessionid" in cookie_names:

            await page.wait_for_timeout(3000)
```

## 9. Technical Specifications

### 9.1 PyO3 Integration

```rust
// crates/ghost-bridge/src/polyglot.rs

use pyo3::prelude::*;
use pyo3::types::PyList;
use ghost_schema::{GhostError, PayloadBlob, RawContext};
use std::path::Path;

/// Polyglot worker loaded from Python
pub struct PolyglotWorker {
    id: String,
    manifest: PolyglotManifest,
    module: Py<PyModule>,
}

impl PolyglotWorker {
    /// Load a polyglot from directory
    pub fn load(path: &Path) -> Result<Self, GhostError> {
        Python::with_gil(|py| {
            // Load manifest
            let manifest_path = path.join("manifest.json");
            let manifest = PolyglotManifest::from_file(&manifest_path.to_string_lossy())?;
            
            // Add to Python path
            let sys = py.import("sys")?;
            let py_path: &PyList = sys.getattr("path")?.downcast()?;
            py_path.insert(0, path.to_string_lossy().to_string())?;
            
            // Add external submodule to path
            let external_path = path.join("..").join("external").join(&manifest.external.path);
            py_path.insert(0, external_path.to_string_lossy().to_string())?;
            
            // Import module
            let module = py.import("wrapper")?;
            
            Ok(Self {
                id: manifest.id.clone(),
                manifest,
                module: module.into(),
            })
        })
    }
    
    /// Execute the polyglot worker
    pub fn execute(&self, ctx: &RawContext) -> Result<PayloadBlob, GhostError> {
        Python::with_gil(|py| {
            // Serialize context
            let ctx_json = serde_json::to_string(ctx)?;
            
            // Call Python function
            let result = self.module.call_method(py, "execute_worker", (ctx_json,))?;
            let result_str: String = result.extract(py)?;
            
            // Parse result
            let payload: PayloadBlob = serde_json::from_str(&result_str)?;
            
            Ok(payload)
        })
    }
}
```

## 10. Health & Routing Configuration

### 10.1 Health Scoring

```rust
/// Health scoring for polyglot workers
pub struct PolyglotHealthScorer {
    success_weight: f64,      // 0.60
    latency_weight: f64,      // 0.25
    quality_weight: f64,      // 0.15
}

impl PolyglotHealthScorer {
    pub fn calculate(&self, metrics: &WorkerMetrics) -> f64 {
        let success_score = metrics.success_rate;
        let latency_score = self.normalize_latency(metrics.avg_latency_ms);
        let quality_score = metrics.data_quality;
        
        (success_score * self.success_weight)
            + (latency_score * self.latency_weight)
            + (quality_score * self.quality_weight)
    }
    
    fn normalize_latency(&self, latency_ms: u64) -> f64 {
        const MIN_MS: f64 = 5000.0;
        const MAX_MS: f64 = 60000.0;
        
        if latency_ms <= MIN_MS as u64 { return 1.0; }
        if latency_ms >= MAX_MS as u64 { return 0.0; }
        
        1.0 - ((latency_ms as f64 - MIN_MS) / (MAX_MS - MIN_MS))
    }
}
```
